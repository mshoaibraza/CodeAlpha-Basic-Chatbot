# Basic Chatbot

A simple rule-based chatbot developed in Python that responds to predefined user inputs using conditional statements. This project is designed for beginners to learn the fundamentals of Python programming, including functions, loops, user input handling, and decision-making structures.

---

## 📌 Project Overview

The Basic Chatbot simulates a simple conversation between a user and a computer program. It recognizes specific keywords or phrases and provides predefined responses. The chatbot continues interacting with the user until the exit command is given.

This project demonstrates how conversational programs work without using Artificial Intelligence or Machine Learning.

---

## 🚀 Features

* Interactive command-line chatbot.
* Responds to common greetings and questions.
* Uses predefined rule-based responses.
* Case-insensitive user input handling.
* Continuous conversation using loops.
* Exit command to end the chat.
* Beginner-friendly and easy to understand.

## 📖 How It Works

1. The chatbot starts and waits for user input.
2. The user types a message.
3. The chatbot checks the input using conditional statements.
4. A predefined response is displayed.
5. The process repeats until the user enters **"bye"**.
6. The chatbot exits the conversation gracefully.

---

## 💬 Supported Commands

| User Input        | Chatbot Response                 |
| ----------------- | -------------------------------- |
| hi                | Hello! How can I help you today? |
| hello             | Hello! How can I help you today? |
| how are you       | I'm fine, thanks for asking!     |
| what is your name | I am a simple Python chatbot.    |
| thanks            | You're welcome!                  |
| thank you         | You're welcome!                  |
| bye               | Goodbye! Have a great day.       |

## ▶️ How to Run

1. Install Python 3 on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python chatbot.py
```

5. Start chatting with the bot.
